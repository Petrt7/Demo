Inspector / 開發者工具

ch02_ex01_inspector.html

Inspector / 開發者工具 是前端 debug 的重要工具。以下內容 Windows 上的 Chrome 為例

快捷鍵: Ctrl + Shift + i

較常用的功能

  • Elements: 檢視網頁的 HTML elements。亦可在此暫時改變 CSS
  • Console: 檢視 Error 跟 Log 訊息。Debug 的主要工具
  • Network: 檢視檔案傳輸的狀態。(https://developer.mozilla.org/en-US/docs/Web/HTTP/Status)
