Array 是 Object 的特例 (subclass/子類別), 語法為:

0 1 2
[ value1, value2, value3 ]

範例

let myFavoriteCreatures = [ "Ape", "Bee", "Cat" ];
console.log(myFavoriteCreatures[1]); // Bee
console.log(myFavoriteCreatures.length); // 3

與物件相比可發現, 陣列的語法中並沒有 key。事實上是陣列的 key 是由 0 開始的整數, 且在語法中被省略了
