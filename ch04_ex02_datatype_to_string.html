字串 / String 的相加

ch04_ex02_datatype_to_string.html

在 Javascript 中, 當字串與非字串相加時, 非字串的資料會先被轉換成字串

字串 + a = console.log(a)
字串 "Hey " + "World" Hey World
數字 "10" + 3 103
布林 "Hey " + true Hey true
未定義 "Hey " + b Hey undefined
陣列 "Hey " + ["Apple", "Banana"] Hey Apple,Banana
物件 "Hey " + { a: 1, b: 2 } Hey [Object object]
