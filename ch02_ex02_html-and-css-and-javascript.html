HTML

HTML: HyperText Markup Language / 超文本標記語言

以下為一 HTML 範例, 反應本範例中標題部分

<div>
    <h1 id="example-title" class="section-title">HTML & CSS & Javascript</h1>
    <p id="example-filename">ch02_ex02_html-css-javascript.html</p>
</div>
  • HTML 的基本語法是將用 opening tag (如<div>, <h1>, <p>) 跟 closing tag (如</div>, </h1>, </p>)包住文本
  • 有些 element 可以包含其他 element。此行為稱為 nesting。如上例中 div 包含了 h1 及 p, 此時該 div 為 h1 及 p 的親元素; h1 及 p 為 div 的子元素; h1 與 p 互為手足元素 (siblings)
<div id="example-info" class="center section">
    <h1>HTML & CSS & Javascript</h1>
    <p>ch02_ex02_html-css-javascript.html</p>
</div>
  • id: 一個元素特有的識別屬性
  • class: 多個元素特有的識別屬性
