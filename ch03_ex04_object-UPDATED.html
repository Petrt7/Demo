資料型態: 物件 / Object

ch03_ex04_object.html

物件 / Object 是一種資料型態, 它能夠同時儲存複數個、不同資料型態的值

物件的例子: 汽車

從實體物件的角度來想像物件, 以一輛車為例:

一輛汽車有各種變數, 稱為「屬性」 (property)
例如: 型號、重量、顏色、哩程

一輛汽車有各種功能, 稱為「方法」(method)
例如: 啟動、前進、煞車、熄火

語法

key value
{ color: white,
miles: 100,
weight: 850kg, }
  • object 的 property 和 method 的形式為 key-value pair
    key 即屬性或方法的名稱, 如 color, miles
    value 即屬性或方法的值, 如 "white", 100 ( 或 method 對應的 function )
  • 名稱與值間以冒號連結
  • key-value pair 間以逗號分開
  • 名稱可以用引號包著, 也可以不用

物件的例子: 幻燈片

另一個符合網站實際應用的物件例子為商業網站上常見到的幻燈片

幻燈片可能有的屬性:
所有圖片的 url
目前在第幾張投影片

幻燈片可能有的方法:
跳到下一張圖片 (在點擊向右的箭頭時呼叫)
跳到前一張圖片 (在點擊向左的箭頭時呼叫)
跳到第 n 張圖片 (在點擊小圓圈時呼叫)
