方法(Method)及屬性(Property)

ch03_ex03_method-and-property.html

方法(Method)

  • String, Number 等 Datatype 有各種預設的 function, 讓我們能對資料進行操作。例如把 String 變成全部大寫, 或把 String 切割
  • 此類 function 稱為「方法 / method」

屬性(Property)

  • String, Number 等 Datatype 有各種預設的變數, 用來表示資料的狀態。例如 String 有多少字數
  • 此類變數稱為「屬性 / property」
