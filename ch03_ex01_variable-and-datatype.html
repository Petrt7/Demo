變數與常見的資料型態

ch03_ex01_variable-and-datatype.html

變數用於儲存資料, 讓該資料能於多處使用, 並在必要時統一更新

  • var: 最早的宣告方式, 最為"鬆散"
  • let: 較晚加入 (ES6, 2015)
  • const: 較晚加入 (ES6, 2015)

Javascript 中常見的資料型態有:

  • 字串 / String: 由雙引號或引號包夾的值。如 "a", 'b', "1", "2", "3"
  • 數字 / Number: 0 到 9 組成的整數，如 1, 2, 3。也可以有小數, 如 0.1, 0.2, 0.3
  • 布林 / Boolean: true 或 false
  • 未定義 / Undefined: 值只有一種, 即 undefined
